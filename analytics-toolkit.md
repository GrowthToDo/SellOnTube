## How to Reuse This Toolkit for a New Website

### Step 1: Google Setup
1. Create a Google Cloud project and enable: Analytics Data API, Search Console API, Indexing API
2. Create a service account and download `credentials.json`
3. Add service account as Viewer in GA4 and Owner in GSC
4. Update `GA4_PROPERTY_ID`, `GSC_SITE_URL`, and `CREDENTIALS_PATH` in `mcp_seo_server.py`

### Step 2: Bing Setup
1. Create a Microsoft account and add site to [Bing Webmaster Tools](https://www.bing.com/webmasters/) (import from GSC is easiest)
2. Generate an API key from Bing Webmaster Settings
3. Add `BING_WEBMASTER_API_KEY` to `.env` and `.mcp.json`

### Step 3: Microsoft Clarity
1. Create a project at [clarity.microsoft.com](https://clarity.microsoft.com/)
2. Add the tracking script to your analytics component (replace project ID)
3. Link to GA4 in Clarity Settings > Setup > Google Analytics integration

### Step 4: IndexNow
1. Generate a key at [indexnow.org](https://www.indexnow.org/)
2. Host the key as a `.txt` file at site root (e.g., `/your-key.txt`)
3. Add `INDEXNOW_KEY` to `.env` and `.mcp.json`

### Step 5: DataForSEO (optional, paid)
1. Sign up at [dataforseo.com](https://dataforseo.com/)
2. Add `DATAFORSEO_LOGIN` and `DATAFORSEO_PASSWORD` to `.env` and `.mcp.json`

### Step 6: Ahrefs (optional, free tier)
1. Sign up for Ahrefs Webmaster Tools at [ahrefs.com/webmaster-tools](https://ahrefs.com/webmaster-tools)
2. Verify site ownership
3. Run crawls and export CSVs for analysis

### Step 7: Site Health Tools
No setup needed. The broken link checker, meta tag auditor, internal link mapper, and schema validator work by crawling the live site. Just update the base URL in `mcp_seo_server.py`.

## Quick Reference: MCP Commands

```
# Traffic & Rankings
get_top_queries          — Google search queries driving traffic
get_ranking_opportunities — Queries ranking position 4-20 (quick wins)
get_top_pages            — Top pages by sessions from GA4
get_traffic_sources      — Channel breakdown (organic, direct, referral)
get_gsc_pages            — Page-level GSC performance

# Bing
bing_top_queries         — Bing/Yahoo/DuckDuckGo/Copilot queries
bing_top_pages           — Top pages on Bing
bing_crawl_stats         — Bingbot crawl statistics

# Indexing
google_indexing_submit   — Submit URLs to Google for fast indexing
indexnow_submit          — Submit URLs to Bing/Yahoo/DuckDuckGo/Yandex

# Keywords (DataForSEO)
dfs_keyword_metrics      — Live search volume, CPC, competition
dfs_serp_results         — Top 10 Google results for a keyword
dfs_keyword_suggestions  — Related keywords for a seed term

# Site Health
pagespeed_check          — Core Web Vitals and performance scores
schema_validate          — JSON-LD structured data validation
broken_link_check        — Find 404s and dead links
meta_tag_audit           — Title/description/OG tag issues
internal_link_map        — Orphan pages and internal link gaps
```
