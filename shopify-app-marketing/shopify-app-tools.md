# shopify-app-tools.md — Microtools / Lead Magnets (STRUCTURE + STORED IDEAS — build DEFERRED)

> Do not build the tools this session. Store the plan. Final 3–5 chosen later, one at a time. Each tool lives at `/shopify-app/tools/[keyword-slug]` (see `shopify-app-architecture.md`).

## Purpose

Each tool gives a Shopify app founder a genuinely useful output, captures email, and bridges to "book a discovery call." The output should also **surface a gap the founder can't fully close alone** — which is what the service sells.

## Approach (decided)

- AI-powered, calling **Gemini Flash 2.5** (founder holds the key). [NEEDS INPUT: confirm exact model string — likely "gemini-2.5-flash".]
- Tool selection is a **swappable slot.** Final 3–5 picked later, one at a time.
- Every tool must map to at least one mission outcome (installs / conversion / retention).

## Candidate tools (decide later)

1. **App Store Listing Grader** — paste listing → score + rewrite suggestions. Pillar: ASO. Outcome: installs.
2. **Listing Title & Subtitle Rewriter** — 5 merchant-intent variants. Pillar: ASO. Outcome: installs.
3. **Churn Cost Calculator** — installs + uninstall rate + price → revenue lost to week-one churn. Pillar: docs/onboarding. Outcome: retention.
4. **SEO Content Idea Generator** — describe app → 10 pre-search blog topics. Pillar: SEO. Outcome: installs/conversion.
5. **Review Reply Writer** — paste review → on-brand reply draft. Pillar: review mgmt. Outcome: conversion/retention.
6. **Pricing Model Recommender** — app + current pricing → structure recommendation. Pillar: pricing. Outcome: conversion.
7. **Knowledge Base Article Generator** — describe feature → draft help article. Pillar: docs. Outcome: retention.
