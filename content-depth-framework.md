# SellOnTube Content Depth Framework

## Core Principle: Depth for Value, Not Word Count

Optimize every SellOnTube page (blog, BOFU, pSEO, service, tool) for:

- Search intent satisfaction
- Buyer/user usefulness
- Practical value
- Semantic completeness
- Extractability for AI/search engines
- Conversion relevance

**Default Stance:** Deeper pages often win by covering subtopics, long-tail queries, buyer questions, use cases, objections, and quotable passages. But length is a byproduct of completeness -- never the goal. Cut anything that doesn't add real value.

---

## Content Depth Decision Tree

Evaluate topic intent, then choose:

| Coverage Level | Word Range | When to Use | Key Traits |
|---|---|---|---|
| Deep | 2,000-4,000+ | Broad/complex intent; buyer decisions; multiple follow-ups; strategic depth; reference potential (e.g., YouTube SEO frameworks for B2B SaaS) | Examples, comparisons, checklists, implementation steps |
| Medium | 1,200-2,200 | Specific intent; practical clarity needed (e.g., "YouTube thumbnail optimization for SaaS demos") | Balanced guide without excess |
| Short | 500-1,200 | Narrow/quick intent (e.g., tool quickstart) | Direct answers, no fluff |

**Hard Rule:** No padding. Skip generic intros, repetition, keyword stuffing, or filler. If 1,100 words nails it, stop. If 3,800 adds true value, expand.

---

## "Comprehensive but Tight" Checklist

Every page must cover the reader's job-to-be-done:

- What is it?
- Why matters?
- Who for/not for?
- When to use?
- How to do it?
- Mistakes to avoid?
- Real examples?
- Next steps?

Include only relevant ones. Aim for self-contained answers.

---

## AI/LLM Readability Rules

Make pages scannable for humans and extractable for Google/AI:

- Clear H2/H3 headings (e.g., "YouTube SEO for SaaS: Thumbnail Best Practices")
- Concise definitions (1 sentence)
- Short paragraphs (3-5 lines)
- Tables/checklists/bullets
- Standalone blocks (e.g., "Top 3 SaaS Video Hooks: 1. Problem-agitate...")
- SaaS/YouTube entities: activation, onboarding, CTR, AVD, conversion funnels
- Target 5+ quotable passages per page for snippets/AI overviews

---

## Page-Type Guidelines

### Blog/Educational/Strategy (1,800-3,500+ words)

Thesis + frameworks + SaaS examples + takeaways (e.g., "B2B YouTube Funnel for 20% CAC Drop").

### BOFU/Commercial (1,500-3,000+ words)

Problem + use cases + outcomes + objections + trust + CTA (e.g., "Why 80% of SaaS Demos Fail on YouTube").

### pSEO (800-2,200 words)

Prioritize uniqueness > length. Add page-specific insights/examples. Kill duplicates -- don't publish thin pages.

### Tool Pages (700-2,000 words)

Tool-first: Usage, examples, outputs, mistakes, FAQs. No blog bloat.

---

## Editorial Checklist (Before Expanding)

Before adding depth to any page, every section must pass:

- [ ] Answers a real question?
- [ ] Boosts decisions/implementation?
- [ ] Adds specificity/proof?
- [ ] Covers search intent without dilution?
- [ ] Useful to buyers/operators?
- [ ] More AI-retrievable?

If no to any, cut.

---

## Final Standard

High-depth, high-utility pages > thin ones. Value per page > word count.
