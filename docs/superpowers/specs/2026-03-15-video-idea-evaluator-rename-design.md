# Design: Rename Topic Evaluator → Video Idea Evaluator

**Date:** 2026-03-15
**Branch:** youtube-video-ideas-evaluator
**Status:** Approved

## Context

Renaming the YouTube Topic Evaluator microtool to "Video Idea Evaluator" to target the
higher-volume keyword cluster around "youtube video ideas" (500 SV, KD 0 per sot_master.csv).
The tool functionality does not change — only the name, URL, metadata, and copy.

## Keyword Rationale

| Keyword | Volume | KD |
|---|---|---|
| youtube video ideas generator | 500 | 0 |
| youtube video ideas for businesses | 0 | 0 |

"generator" is reserved for a separate planned tool. Meta title uses "Evaluator" framing
to stay accurate to the tool's function while capturing the "video ideas" keyword cluster.
