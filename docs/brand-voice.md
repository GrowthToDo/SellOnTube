# SellOnTube Brand Voice — Quick Reference

> This is a pointer file. Full rules live in the source docs below. Read those before writing.

---

## Voice in one sentence

Data-driven, direct, founder-to-founder. No fluff, no hedging, no consultant-speak.

## Tone spectrum

| Always | Sometimes | Never |
|---|---|---|
| Confident, specific | Opinionated, contrarian | Hedging, vague |
| Practical ("here's how") | Blunt ("this doesn't work") | Academic, theoretical |
| B2B acquisition lens | Industry-specific angles | Creator/influencer framing |
| Short punchy + longer depth | One-sentence paragraphs | Uniform paragraph lengths |

## The ICP filter

Every sentence must answer: "Why does this matter for a business acquiring customers?"

If it could appear on VidIQ or TubeBuddy without modification, reframe it. See `docs/icp.md` for the full ICP definition.

## Banned patterns (quick checklist)

- Em-dashes (`—`) — use comma, colon, or restructure
- "Most [topic]..." openers
- AI vocabulary: delve, pivotal, tapestry, landscape (abstract), showcase, foster, vibrant, intricate
- Hedging: "it might be", "could potentially", "it is possible that"
- Filler: "In today's digital landscape", "Now more than ever"
- Jargon the ICP wouldn't search: "High-LTV", "Compounding Flywheel"

Full banned list: `style-guide.md` Fix #2 + Fix #3.
