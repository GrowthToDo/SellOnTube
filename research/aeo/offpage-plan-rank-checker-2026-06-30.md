# Off-Page Plan — Rank-Checker Cluster Authority (2026-06-30)

## Why this, why now
GSC: the "youtube rank checker / rank tracker" cluster = ~40k impressions/90d (63% of all site impressions), stuck at **pos 30-55**. On-page is already strong (deep content, full schema, 33 internal links, in nav). The recon + SERP diagnosis both point to the same ceiling: **domain authority**. Page 1 is held by dedicated tool pages (TubeLab, TubePilot, TubeRanker, Lenos, Seomator, Wiz, views4you) plus DA giants (Semrush, SocialBlade). The only lever left is **links to the free tool**, which is a genuine linkable asset (free, no signup, returns top-20 + competitors).

Target asset: `https://sellontube.com/tools/youtube-ranking-checker` (and the cluster posts as secondary).

Honest expectation: this is the slowest lever (weeks to months) and is mostly **off-code** (accounts, email, forms). It is also the only one that moves the 40k impressions. Everything in here needs a human to execute; this doc is the target list + templates.

## Track 1 — Free-tool directories (fastest, lowest effort)
Submit the free Ranking Checker (and the other free tools) as listings. High-yield, mostly free, no relationship needed.

Priority targets:
- **There's An AI For That** / AI-tool directories (tool has a Gemini-style angle on other tools; rank-checker is a straight utility, list as "free YouTube tool").
- **SaaSHub**, **Product Hunt** (launch the tool/tool-suite), **AlternativeTo** (list as alternative to TubeBuddy/vidIQ rank tracking), **G2 / Capterra** (free-tool listing if eligible).
- **Free SEO tool roundups/directories**: Detailed.com-style tool lists, BuiltWith tool listings, Indie Hackers "tools", **Toolify**, **Futurepedia**.
- YouTube-creator tool lists: communities + resource pages that aggregate free creator tools.

Action: one owner, one week, batch-submit. Track in a sheet (directory, URL submitted, status, live link, date). This connects to the paused 72-directory plan in memory (`project_directory_submissions`, Phase 2 needs a browser).

## Track 2 — Roundup placement (highest link quality)
The listicles already ranking for our target terms are link prospects: get the free tool **added** to them.

Prospect list (from the live SERP for "youtube rank checker/tracker" + "best youtube tools"):
- Roundups that list TubeLab / TubePilot / TubeRanker / Lenos / Seomator / views4you — pitch "add a free, no-signup, business-focused option."
- "Best free YouTube SEO tools 2026" / "best YouTube rank checker" listicles on mid-DA marketing blogs (limelightdigital, mayple-type sites surfaced in the autocomplete SERP).
- Our own `best-youtube-rank-checker-tools-for-business` ranks for these terms; mirror its competitor set to find every roundup that should include us.

Pitch angle: most listed tools require signup or are creator-focused. SellOnTube is **free, no login, returns top-20 + competitor view counts, business-framed**. That's a genuine differentiator a roundup editor can use.

## Track 3 — Link-intersect (steal competitor links)
Find sites linking to TubeLab/checker, TubePilot, TubeRanker but not to us. Use Ahrefs (already in the stack) Link Intersect: targets = those 3-4 tool domains, then filter for resource/roundup pages. Pitch the same differentiator. This is the most systematic source.

## Track 4 — Community + digital PR (slower, compounding)
- **Reddit**: per `docs/strategy/reddit-marketing-playbook.md` + agent 10. Subreddits where people ask "how do I check my YouTube rank" (r/youtubers, r/NewTubers, r/PartneredYoutube). Answer genuinely, link the free tool when it actually helps. No spam.
- **HARO / Qwoted / Featured**: respond to "YouTube SEO / creator tools" queries; earn a brand mention + link.
- **Comparison content**: a "free vs paid YouTube rank trackers" piece that earns links by being genuinely useful (we already have the cluster content to support it).

## Sequencing (recommended)
1. Week 1: Track 1 directories (batch) + Product Hunt launch prep.
2. Week 1-2: Track 3 link-intersect pull in Ahrefs → prospect sheet.
3. Week 2-4: Track 2 roundup outreach (10-20 pitches) + Track 4 Reddit/HARO ongoing.
4. Re-pull GSC at ~4 and ~8 weeks; watch the cluster's avg position, not just clicks (position moves first).
